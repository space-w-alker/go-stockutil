About go-owndoc

go-owndoc is a project that aims to provide a clean, flexible, and customizable mechanism for generating and self-hosting documentation and API reference for Golang projects.

It uses the same underlying mechanisms as the godoc command, but offers much greater ability to customize the output of the documentation. The aim is to allow projects, especially private repositories, to generate and host their own high-quality updated code documentation.


This project is built and maintained by Gary Hetzel, and the source code may be viewed, copied, and modified on GitHub.
